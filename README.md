## 📋 1. Contexto e Objetivos

Este repositório faz parte de um desafio de projeto na DIO. O objetivo principal é explorar a sinergia entre a linguagem **Python** e a ferramenta de automação **n8n**. ## 📖 5. Miniguia de Estudo (Entrega Final)

### 🧾 Resumo Estruturado

Python é uma linguagem de programação de script interpretada e multiparadigma, ela foi concebida para ser simples de entender e possuir uma biblioteca padrão bastante abrangente, é uma escolha frequente para processamento de dados avançado, análise de dados e web scraping. Características que facilitam seu uso Simplicidade de sintaxe, multiplataforma, disponibilidade de bibliotecas. O n8n é uma plataforma de automação de fluxos de trabalho (workflows) que se destaca por permitir a integração de diversos serviços e aplicativos com pouco ou nenhum uso de código. Suas funcionalidades principais giram em torno da orquestração de processos de forma visual e flexível. O n8n pode orquestrar o fluxo, enquanto o Python lida com a manipulação pesada de strings ou APIs específicas.

### 🧠 Glossário de Termos-Chave

#### ⚙️ **Conceitos de Automação**

- Automação: uso de tecnologia para executar tarefas repetitivas automaticamente.
- Workflow: sequência de etapas automatizadas dentro de um processo, como um fluxo de ações conectadas.
- Node: bloco dentro do n8n que representa uma ação, como enviar um e-mail ou processar dados.
- Trigger: elemento que inicia uma automação (ex: horário, webhook ou evento).
- Low-code / No-code: desenvolvimento com pouco ou nenhum código, utilizando interfaces visuais, mas que permite adicionar código quando necessário.

---

#### 🔗 **Ferramentas e Integração**

- Python: linguagem de programação utilizada para criar scripts e automações.
- n8n: ferramenta de automação que permite criar fluxos integrando diferentes sistemas.
- API: forma de comunicação entre sistemas para troca de dados.
- Webhook: mecanismo que envia dados automaticamente quando um evento acontece.

---

#### 🐍 **Conceitos básicos de Python**

- int: números inteiros (ex: 10, -50).
- float: números com casas decimais (ex: 3.14).
- str: textos entre aspas.
- bool: valores lógicos (True ou False).
- list: coleção de dados que pode ser alterada.
- tuple: coleção de dados imutável.
- dict: dados em formato chave e valor.
- set: conjunto de valores únicos.
- len(): função que retorna o tamanho de um dado.
- split(): divide um texto em partes.
- join(): junta elementos em um único texto.
- indentação: recuo do código que define a estrutura lógica no Python.

---

#### 💻 **Execução e Infraestrutura**

- REPL: modo interativo do Python que executa comandos em tempo real.
- Modo Batch: execução de scripts completos a partir de arquivos.
- Docker: tecnologia que cria ambientes isolados para execução de aplicações.
- VPS: servidor virtual que roda aplicações continuamente na nuvem.
